# 高清电影命名解析

## demo / example

[电影命名解析](https://bd.xinshangshangxin.com)

## 介绍

一般来说，正规压制组压制的电影，都采用 0day 命名方式，即：英文名称、版本说明、年份、片源、分辨率、视频编码、音频格式、压制小组。 例如文件名：Jumanji.The.Next.Level.DC.2019.BluRay.1080p.X264.DTS-HD.MA5.1-MTeam。

## 作用

解析文件命名各个含义
